Base R statistical analysis examples
====================================

This ZIP contains standalone R scripts using base R only. They use either
simulated data or built-in functionality, so they do not require installing
external packages.

Run one script from a shell with, for example:

    Rscript 10_simple_linear_regression.R

Or from inside R:

    source("10_simple_linear_regression.R")
